## Rendering rules

- Smooth soft shading (Lambert), NO faceted low-poly, NO outlines.
- Buildings: simple rounded-feeling boxes; mostly white/cream bodies,
  ~1 in 3 gets a colored roof slab; occasional second-tier box.
- Roads: real OSM geometry (`web/roads.json`), flat ribbons; widths by
  class (primary 24u ... residential 11u).
- Density: urban zones cluster along road frontage; green zones get
  grass + trees; zone centers stay clear (gathering plazas, r=60u).
- Sky: gradient dome + solid cel cumulus (flat bases, blue under-shadow).
- Scale: toy-town — characters deliberately oversized (CHAR_H 15u ≈ a
  2-storey building), per the reference's composition.

## Generation recipe (for new reference art)

Every new reference image (landmarks, districts, night variant, seasonal)
must be generated by RE-ANCHORING to the master reference — attach
`world-style-locked_globe-smooth.png` as the style/reference image in
Higgsfield each time. Never chain off a previous generation (drift).

Prompt template (fill the bracket, keep the rest verbatim):

> [SUBJECT — e.g. "a landmark orange-roofed arcade building"], smooth
> soft-shaded 3D toy-town style, rounded simple geometry, white and cream
> buildings with occasional coral/teal/yellow accents, bright saturated
> blue sky with puffy white clouds, curved globe horizon, green grass
> fields, cute miniature diorama proportions, clean and uncluttered,
> no text or signage, no outlines, no low-poly faceting
